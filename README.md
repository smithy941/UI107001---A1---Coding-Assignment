# Algorithms and Data Structures Assignment

This project contains algorithms and data structures with unit tests written using Jest.

## How to Run the Tests

1. Open terminal in the project folder

2. Install dependencies:

   npm install

3. Run the tests:

   npm test

## Expected Output

You should see:

PASS ./mergeSort.test.js  
PASS ./binarySearch.test.js  
PASS ./linkedList.test.js  
PASS ./hashMap.test.js  

Test Suites: 4 passed, 4 total  
Tests:       17 passed, 17 total  

## Notes

- All tests are written using Jest  
- Tests cover correct behaviour, edge cases and boundary conditions  
- The node_modules folder is not included and will be created after running npm install
